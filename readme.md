# Discord Scheduler Bot

A feature-rich Discord bot for scheduling recurring notifications and announcements with comprehensive logging and persistence.

## Features

- 📅 Multiple schedule types:
  - Daily (with custom frequency)
  - Weekly
  - Biweekly
  - Monthly
  - Custom (using cron expressions)
- 📝 Detailed logging system
- 💾 Persistent schedule storage
- 🎯 Role-based notifications
- ⚡ Slash command interface
- 🔍 Schedule management commands

## Setup

### Prerequisites

- Node.js v16.9.0 or higher
- npm (Node Package Manager)
- A Discord Bot Token
- A Discord Application with slash commands enabled
